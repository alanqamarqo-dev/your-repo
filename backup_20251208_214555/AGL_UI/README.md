AGL Local UI

A tiny Tkinter-based GUI to run AGL locally and view the generated report.

Run:

```bash
python AGL_UI/main.py
```

Features:

- Run a task via the Run button (spawns AGL.py to write `reports/last_run.json`).
- Refresh and display the last JSON report.

This is intentionally minimal for development/demo use inside VS Code.
