6-Layer Security Pipeline

Each layer adds depth to the analysis, from fast pattern matching to mathematical proof of exploitability.

🔍
Layer 1

Pattern Engine

Fast regex-based vulnerability pattern scanning across the entire codebase. Catches common issues like reentrancy patterns, unchecked calls, and dangerous delegatecall usage.

🔬
Layer 2

22 Specialized Detectors

Deep AST-based analysis with 22 purpose-built detectors covering reentrancy, access control, integer overflow, flash loan attacks, price manipulation, and more.

🧮
Layer 3

Z3 Symbolic Engine

Microsoft Z3 theorem prover formally verifies vulnerability conditions. Produces mathematical proof that a bug is real — not just a heuristic guess.

⚔️
Layer 4

Exploit Reasoning

4-stage pipeline: Path Extraction → Constraint Solving → Invariant Checking → Exploit Assembly. Generates concrete exploit proofs with step-by-step attack sequences.

📊
Layer 5

RiskCore Scoring

Logistic regression model combining heuristic confidence, formal proof scores, and prior probability to compute calibrated exploit probability for each finding.

🌊
Layer 6

Heikal Mathematics

Advanced quantum tunneling scoring and wave domain evaluation. Novel mathematical framework for assessing vulnerability reachability through complex contract interactions.

Quick Start

Installation & Usage 
# Clone the repository — استنساخ المستودع
git clone https://github.com/alanqamarqo-dev/your-repo.git
cd your-repo

# Install dependencies — تثبيت المتطلبات
pip install -r requirements.txt

# Run a security audit — تشغيل تدقيق أمني
python3 scripts/run_full_audit.py --target contracts/MyContract.sol

# Output: reports/FULL_AUDIT_REPORT.md — الناتج: تقرير كامل

Requirements

  • Python 3.10+
  • Z3 Solver (pip install z3-solver)
  • solc 0.8.x (solc-select)
  • Optional: Slither, Mythril for enhanced analysis
