# System Monitoring / مراقبة النظام

## English

Tools for observing the health and performance of the AGL system in real-time.

**Key Components:**

- `Performance_Tracker.py`: Logs execution times and resource usage.
- `Error_Logger.py`: Captures and categorizes errors.
- `Dashboard_Feeder.py`: Prepares data for the web dashboard.

## العربية

أدوات لمراقبة صحة وأداء نظام AGL في الوقت الفعلي.

**المكونات الرئيسية:**

- `Performance_Tracker.py`: يسجل أوقات التنفيذ واستخدام الموارد.
- `Error_Logger.py`: يلتقط ويصنف الأخطاء.
- `Dashboard_Feeder.py`: يجهز البيانات للوحة معلومات الويب.
