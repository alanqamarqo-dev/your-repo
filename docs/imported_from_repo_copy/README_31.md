# Tools / الأدوات

## English

A collection of utility scripts and tools for testing, evaluation, data processing, and system maintenance.

**Key Components:**

- `agi_full_test.py`: Comprehensive test suite for AGI capabilities.
- `evaluate_rag.py`: Tools for evaluating Retrieval-Augmented Generation performance.
- `probe_llm.py`: Scripts for probing LLM behavior and outputs.
- `generate_pdf_report.py`: Generates formatted PDF reports of system status.

## العربية

مجموعة من النصوص البرمجية والأدوات المساعدة للاختبار، والتقييم، ومعالجة البيانات، وصيانة النظام.

**المكونات الرئيسية:**

- `agi_full_test.py`: مجموعة اختبار شاملة لقدرات AGI.
- `evaluate_rag.py`: أدوات لتقييم أداء التوليد المعزز بالاسترجاع (RAG).
- `probe_llm.py`: نصوص برمجية لفحص سلوك ومخرجات LLM.
- `generate_pdf_report.py`: يولد تقارير PDF منسقة لحالة النظام.
