# Dynamic Modules / الوحدات الديناميكية

## English

Modules that are designed to be loaded dynamically or on-demand. This includes specialized solvers, games, and experimental features.

**Key Components:**

- `mission_control.py`: The central orchestrator for complex missions.
- `unified_agi_system.py`: An integrated system combining multiple capabilities.
- `snake_game.py`: A simple game for testing interactive capabilities.
- `complex_math_solver.py`: Advanced mathematical problem solving.

## العربية

وحدات مصممة ليتم تحميلها ديناميكيًا أو عند الطلب. يشمل ذلك الحلول المتخصصة، والألعاب، والميزات التجريبية.

**المكونات الرئيسية:**

- `mission_control.py`: المنسق المركزي للمهام المعقدة.
- `unified_agi_system.py`: نظام متكامل يجمع بين قدرات متعددة.
- `snake_game.py`: لعبة بسيطة لاختبار القدرات التفاعلية.
- `complex_math_solver.py`: حل مسائل رياضية متقدمة.
