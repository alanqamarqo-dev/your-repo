# Core Consciousness / الوعي الأساسي

## English

Modules responsible for the system's self-awareness and perception.

**Key Components:**

- `Self_Model.py`: Maintains the system's understanding of itself.
- `Perception_Loop.py`: Processes inputs and updates internal state.
- `State_Logger.py`: Records the stream of consciousness.
- `Intent_Generator.py`: Formulates high-level goals and intentions.

## العربية

الوحدات المسؤولة عن الوعي الذاتي للنظام وإدراكه.

**المكونات الرئيسية:**

- `Self_Model.py`: يحافظ على فهم النظام لنفسه.
- `Perception_Loop.py`: يعالج المدخلات ويحدث الحالة الداخلية.
- `State_Logger.py`: يسجل تيار الوعي.
- `Intent_Generator.py`: يصيغ الأهداف والنوايا عالية المستوى.
