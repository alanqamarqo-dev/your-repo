# Knowledge Base / قاعدة المعرفة

## English

This directory stores the static and dynamic knowledge of the AGL system. It acts as the long-term memory and reference library.

**Key Components:**

- `Knowledge_Graph.py`: Manages relationships between concepts.
- `Law_Facts.py` & `law_facts.json`: Stores legal axioms and rules.
- `Learned_Patterns.json`: Accumulates patterns recognized over time.
- `Domain_Expertise.py`: Specialized knowledge modules.

## العربية

يخزن هذا الدليل المعرفة الثابتة والديناميكية لنظام AGL. يعمل كذاكرة طويلة المدى ومكتبة مرجعية.

**المكونات الرئيسية:**

- `Knowledge_Graph.py`: يدير العلاقات بين المفاهيم.
- `Law_Facts.py` و `law_facts.json`: يخزن البديهيات والقواعد القانونية.
- `Learned_Patterns.json`: يراكم الأنماط التي تم التعرف عليها بمرور الوقت.
- `Domain_Expertise.py`: وحدات المعرفة المتخصصة.
