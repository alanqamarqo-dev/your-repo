# Utilities / الأدوات المساعدة

## English

Helper functions and modules used across the AGL system.

**Key Components:**

- `llm_tools.py`: Utilities for interacting with Large Language Models.
- `quantum_adapter.py`: Helper functions for quantum computing simulations.

## العربية

وظائف ووحدات مساعدة تستخدم عبر نظام AGL.

**المكونات الرئيسية:**

- `llm_tools.py`: أدوات مساعدة للتفاعل مع نماذج اللغة الكبيرة.
- `quantum_adapter.py`: وظائف مساعدة لمحاكاة الحوسبة الكمومية.
