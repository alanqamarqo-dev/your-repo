# Config / التكوين

## English

Configuration files that control the behavior of the AGL engines and learning systems.

**Key Files:**

- `engine_preference.py`: Defines which LLM engines are preferred for specific tasks.
- `fusion_weights.json`: Controls how outputs from multiple models are combined.
- `coverage_boost.json`: Settings to improve test coverage or knowledge retrieval.

## العربية

ملفات التكوين التي تتحكم في سلوك محركات AGL وأنظمة التعلم.

**الملفات الرئيسية:**

- `engine_preference.py`: يحدد محركات LLM المفضلة لمهام محددة.
- `fusion_weights.json`: يتحكم في كيفية دمج المخرجات من نماذج متعددة.
- `coverage_boost.json`: إعدادات لتحسين تغطية الاختبار أو استرجاع المعرفة.
