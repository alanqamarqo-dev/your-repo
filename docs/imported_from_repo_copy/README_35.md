# Workers / العاملون

## English

Background worker processes that perform long-running tasks such as knowledge harvesting and reasoning cycles.

**Key Components:**

- `knowledge_harvester.py`: Continuously gathers and processes new information.
- `reasoning_cycle.py`: Runs background reasoning loops to refine knowledge.

## العربية

عمليات العامل في الخلفية التي تؤدي مهام طويلة الأمد مثل حصاد المعرفة ودورات التفكير.

**المكونات الرئيسية:**

- `knowledge_harvester.py`: يجمع ويعالج المعلومات الجديدة باستمرار.
- `reasoning_cycle.py`: يشغل حلقات تفكير في الخلفية لتحسين المعرفة.
