# AGL Local UI / واجهة المستخدم المحلية لـ AGL

## English

A tiny Tkinter-based GUI to run AGL locally and view the generated report.

**Run:**

```bash
python AGL_UI/main.py
```

**Features:**

- Run a task via the Run button (spawns AGL.py to write `reports/last_run.json`).
- Refresh and display the last JSON report.

This is intentionally minimal for development/demo use inside VS Code.

## العربية

واجهة رسومية صغيرة تعتمد على Tkinter لتشغيل AGL محليًا وعرض التقرير الذي تم إنشاؤه.

**التشغيل:**

```bash
python AGL_UI/main.py
```

**الميزات:**

- تشغيل مهمة عبر زر التشغيل (يطلق AGL.py لكتابة `reports/last_run.json`).
- تحديث وعرض آخر تقرير JSON.

تم تصميم هذا ليكون بسيطًا بشكل متعمد لأغراض التطوير/العرض التوضيحي داخل VS Code.
