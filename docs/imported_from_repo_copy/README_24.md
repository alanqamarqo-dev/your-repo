# Safety Control / التحكم في الأمان

## English

The safety enforcement layer of AGL. Ensures that the system operates within defined boundaries.

**Key Components:**

- `Safety_Control_Layer.py`: The main safety enforcement module.
- `Safe_Rollback_System.py`: Reverts changes if safety checks fail.
- `Hierarchical_Control_System.py`: Manages permissions and authority levels.
- `Dialogue_Safety.py`: Filters unsafe or inappropriate content in conversations.

## العربية

طبقة فرض الأمان في AGL. تضمن أن النظام يعمل ضمن الحدود المحددة.

**المكونات الرئيسية:**

- `Safety_Control_Layer.py`: وحدة فرض الأمان الرئيسية.
- `Safe_Rollback_System.py`: يعيد التغييرات إذا فشلت فحوصات الأمان.
- `Hierarchical_Control_System.py`: يدير الأذونات ومستويات السلطة.
- `Dialogue_Safety.py`: يصفي المحتوى غير الآمن أو غير المناسب في المحادثات.
