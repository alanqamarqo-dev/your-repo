# Web Interface / واجهة الويب

## English

Web-based dashboards and user interfaces for interacting with AGL.

**Key Components:**

- `index.html`: The main entry point for the web dashboard.
- `admin_harvest_dashboard.html`: Dashboard for monitoring knowledge harvesting.
- `test_ui.html`: Interface for testing system features.

## العربية

لوحات معلومات وواجهات مستخدم تعتمد على الويب للتفاعل مع AGL.

**المكونات الرئيسية:**

- `index.html`: نقطة الدخول الرئيسية للوحة معلومات الويب.
- `admin_harvest_dashboard.html`: لوحة معلومات لمراقبة حصاد المعرفة.
- `test_ui.html`: واجهة لاختبار ميزات النظام.
