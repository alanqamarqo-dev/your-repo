# Learning System / نظام التعلم

## English

The adaptive heart of AGL. This module enables the system to learn from experience, optimize its own code, and reason about complex problems.

**Key Components:**

- `Self_Learning.py`: The main loop for acquiring new capabilities.
- `Inference_Engine.py`: Performs logical deduction and reasoning.
- `Feedback_Analyzer.py`: Evaluates the success of actions to guide learning.
- `Self_Optimizer.py`: Tunes internal parameters for better performance.

## العربية

القلب المتكيف لـ AGL. تمكن هذه الوحدة النظام من التعلم من التجربة، وتحسين الكود الخاص به، والتفكير في المشكلات المعقدة.

**المكونات الرئيسية:**

- `Self_Learning.py`: الحلقة الرئيسية لاكتساب قدرات جديدة.
- `Inference_Engine.py`: يقوم بالاستنتاج المنطقي والتفكير.
- `Feedback_Analyzer.py`: يقيم نجاح الإجراءات لتوجيه التعلم.
- `Self_Optimizer.py`: يضبط المعلمات الداخلية لأداء أفضل.
