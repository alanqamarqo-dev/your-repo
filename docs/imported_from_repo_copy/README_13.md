# Data / البيانات

## English

Persistent storage for the AGL system. This folder holds the "long-term memory" that persists across restarts.

**Contents:**

- `memory.sqlite`: Structured database for transactional memory.
- `experiences.jsonl`: Log of past interactions and outcomes.
- `problems.jsonl`: Record of encountered problems and their solutions.
- `training/`: Datasets used for fine-tuning or few-shot learning.

## العربية

التخزين الدائم لنظام AGL. يحتوي هذا المجلد على "الذاكرة طويلة المدى" التي تستمر عبر عمليات إعادة التشغيل.

**المحتويات:**

- `memory.sqlite`: قاعدة بيانات منظمة للذاكرة المعاملاتية.
- `experiences.jsonl`: سجل للتفاعلات والنتائج السابقة.
- `problems.jsonl`: سجل للمشكلات التي تمت مواجهتها وحلولها.
- `training/`: مجموعات البيانات المستخدمة للضبط الدقيق أو التعلم بلقطات قليلة.
