# Core Engines / المحركات الأساسية

## English

This directory contains the specialized cognitive modules (Engines) that power the AGL system. Each engine is designed to handle a specific type of task or reasoning.

**Key Engines:**

- `Hosted_LLM.py`: The central interface to the Large Language Model (Ollama/OpenAI).
- `Volition_Engine.py`: Generates internal drives, goals, and curiosity.
- `Recursive_Improver.py`: Enables the system to read and improve its own code.
- `Dreaming_Cycle.py`: Manages memory consolidation and synthetic experience generation during idle times.
- `Mathematical_Brain.py`: Handles complex calculations and formal logic.
- `Web_Search_Engine.py`: Interface for retrieving real-time information from the web.
- `Code_Generator.py`: Specialized in writing and debugging Python code.

## العربية

يحتوي هذا الدليل على الوحدات المعرفية المتخصصة (المحركات) التي تشغل نظام AGL. تم تصميم كل محرك للتعامل مع نوع معين من المهام أو التفكير.

**المحركات الرئيسية:**

- `Hosted_LLM.py`: الواجهة المركزية لنموذج اللغة الكبير (Ollama/OpenAI).
- `Volition_Engine.py`: يولد الدوافع الداخلية والأهداف والفضول.
- `Recursive_Improver.py`: يمكن النظام من قراءة وتحسين الكود الخاص به.
- `Dreaming_Cycle.py`: يدير دمج الذاكرة وتوليد الخبرة الاصطناعية أثناء أوقات الخمول.
- `Mathematical_Brain.py`: يتعامل مع الحسابات المعقدة والمنطق الرسمي.
- `Web_Search_Engine.py`: واجهة لاسترجاع المعلومات في الوقت الفعلي من الويب.
- `Code_Generator.py`: متخصص في كتابة وتصحيح كود Python.
