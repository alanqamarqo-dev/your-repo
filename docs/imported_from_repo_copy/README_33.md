# Volition Engine / محرك الإرادة

## English

The motivational core of the agent. It generates goals, manages curiosity, and drives the agent to act even without external commands.

**Key Components:**

- `Volition_Core.py`: The main logic for generating drives.
- `Goal_Manager.py`: Prioritizes and tracks active goals.
- `Curiosity_Module.py`: Identifies gaps in knowledge and generates queries to fill them.

## العربية

الجوهر التحفيزي للوكيل. يولد الأهداف، ويدير الفضول، ويدفع الوكيل للعمل حتى بدون أوامر خارجية.

**المكونات الرئيسية:**

- `Volition_Core.py`: المنطق الرئيسي لتوليد الدوافع.
- `Goal_Manager.py`: يحدد أولويات ويتتبع الأهداف النشطة.
- `Curiosity_Module.py`: يحدد الفجوات في المعرفة ويولد استفسارات لملئها.
