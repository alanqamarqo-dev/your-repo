# Artifacts / المخرجات (Artifacts)

## English

This directory serves as the output dump for AGL's operations. It contains logs, generated reports, code snippets created by the AI, and session state snapshots.

**Contents:**

- `logs/`: Execution logs.
- `reports/`: Structured JSON reports of missions.
- `codegen_*.py`: Python files generated by the Recursive Improver.
- `session_state_*.json`: Snapshots of the system's memory and state.

**Note:** This folder is critical for debugging and auditing the system's "thought process".

## العربية

يعمل هذا الدليل كمكب للمخرجات لعمليات AGL. يحتوي على السجلات، والتقارير التي تم إنشاؤها، ومقتطفات الكود التي أنشأها الذكاء الاصطناعي، ولقطات لحالة الجلسة.

**المحتويات:**

- `logs/`: سجلات التنفيذ.
- `reports/`: تقارير JSON منظمة للمهام.
- `codegen_*.py`: ملفات Python التي تم إنشاؤها بواسطة المحسن التكراري.
- `session_state_*.json`: لقطات لذاكرة النظام وحالته.

**ملاحظة:** هذا المجلد ضروري لتصحيح الأخطاء وتدقيق "عملية التفكير" للنظام.
