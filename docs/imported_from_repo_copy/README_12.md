# Core Memory / الذاكرة الأساسية

## English

The bridge between the cognitive engines and the storage systems.

**Key Components:**

- `Conscious_Bridge.py`: Connects consciousness to memory.
- `Smart_Protocol_Engine.py`: Manages data retrieval protocols.
- `memory.db`: Local SQLite database for immediate memory access.

## العربية

الجسر بين المحركات المعرفية وأنظمة التخزين.

**المكونات الرئيسية:**

- `Conscious_Bridge.py`: يربط الوعي بالذاكرة.
- `Smart_Protocol_Engine.py`: يدير بروتوكولات استرجاع البيانات.
- `memory.db`: قاعدة بيانات SQLite محلية للوصول الفوري للذاكرة.
