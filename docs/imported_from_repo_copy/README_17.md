# Infrastructure / البنية التحتية

## English

Core infrastructure components for system monitoring, safety enforcement, and adaptive learning.

**Key Components:**

- `monitoring/`: System health and performance monitoring.
- `safety.py`: Safety checks and constraints.
- `adaptive/`: Modules for adaptive learning rates and strategies.
- `harvester/`: Tools for gathering data from external sources.

## العربية

مكونات البنية التحتية الأساسية لمراقبة النظام، وفرض الأمان، والتعلم التكيفي.

**المكونات الرئيسية:**

- `monitoring/`: مراقبة صحة وأداء النظام.
- `safety.py`: فحوصات وقيود الأمان.
- `adaptive/`: وحدات لمعدلات واستراتيجيات التعلم التكيفي.
- `harvester/`: أدوات لجمع البيانات من مصادر خارجية.
