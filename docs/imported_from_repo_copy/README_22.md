# Quality Assurance / ضمان الجودة

## English

Modules for assessing and improving the quality of system outputs.

**Key Components:**

- `ar_answer_quality.py`: Evaluates the quality of Arabic responses.
- `text_postprocess.py`: Cleans and formats text outputs.

## العربية

وحدات لتقييم وتحسين جودة مخرجات النظام.

**المكونات الرئيسية:**

- `ar_answer_quality.py`: يقيم جودة الإجابات العربية.
- `text_postprocess.py`: ينظف وينسق المخرجات النصية.
