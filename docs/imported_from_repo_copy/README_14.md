# Domain Templates / قوالب المجال

## English

JSON templates defining the structure and initial knowledge for specific scientific domains.

**Files:**

- `chemistry.json`: Template for chemical knowledge.
- `physics.json`: Template for physical laws and constants.
- `quantum.json`: Template for quantum mechanics concepts.

## العربية

قوالب JSON تحدد الهيكل والمعرفة الأولية لمجالات علمية محددة.

**الملفات:**

- `chemistry.json`: قالب للمعرفة الكيميائية.
- `physics.json`: قالب للقوانين والثوابت الفيزيائية.
- `quantum.json`: قالب لمفاهيم ميكانيكا الكم.
