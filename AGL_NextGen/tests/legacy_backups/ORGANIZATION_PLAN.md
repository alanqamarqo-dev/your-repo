# خطة تنظيم مشروع AGL (AGL Project Organization Plan)

لضمان عدم كسر النظام (To ensure the system doesn't break), سنقوم بالتنظيم على مراحل.

## الهيكل المقترح (Proposed Structure)

```
d:\AGL\
├── 📂 docs/                  # التوثيق والتقارير (Documentation & Reports)
│   ├── *.md                  # (نقل ملفات الماركدون ما عدا README)
│   └── *.txt                 # (ملفات النصوص والسجلات)
│
├── 📂 results/               # النتائج والمخرجات (Results & Artifacts)
│   ├── *.json                # (نتائج الاختبارات)
│   ├── *.png                 # (الصور والمخططات)
│   ├── *.wav                 # (الملفات الصوتية)
│   └── *.npy, *.pt           # (ملفات البيانات والنماذج)
│
├── 📂 tests/                 # الاختبارات (Tests)
│   ├── test_*.py             # (اختبارات بايثون القياسية)
│   └── *_TEST.py             # (اختبارات النظام المحددة)
│
├── 📂 scripts/               # نصوص التشغيل (Execution Scripts)
│   ├── run_*.py              # (سكربتات التشغيل)
│   ├── activate_*.py         # (سكربتات التفعيل)
│   ├── simulate_*.py         # (سكربتات المحاكاة)
│   └── demo_*.py             # (العروض التوضيحية)
│
├── 📂 src/                   # الكود المصدري الأساسي (Core Source Code)
│   ├── agl_core/             # (نقل ملفات AGL_*.py الأساسية هنا لاحقاً)
│   └── ...
│
└── 📄 README.md              # (يبقى في الجذر)
└── 📄 requirements.txt       # (يبقى في الجذر)
└── 📄 .gitignore             # (يبقى في الجذر)
```

## خطوات التنفيذ (Execution Steps)

1.  **المرحلة الأولى (آمنة جداً):** إنشاء المجلدات ونقل ملفات التوثيق والنتائج (`docs`, `results`). هذا لن يؤثر على عمل الكود.
2.  **المرحلة الثانية (آمنة نسبياً):** نقل ملفات الاختبار (`tests`). قد نحتاج لتعديل مسارات الاستيراد داخل ملفات الاختبار.
3.  **المرحلة الثالثة (حساسة):** نقل نصوص التشغيل (`scripts`). ستحتاج لإضافة مسار الجذر إلى `sys.path` لكي تعمل.
4.  **المرحلة الرابعة (عالية الخطورة):** نقل ملفات الكود الأساسي (`src`). تتطلب تحديث كل جمل `import` في المشروع.

## توصيتي للبدء (Recommendation)

أقترح أن نبدأ **بالمرحلة الأولى** (نقل التقارير والنتائج) لتنظيف المجلد الرئيسي، ثم ننتقل للمراحل الأخرى بحذر.

هل توافق على البدء بإنشاء المجلدات ونقل ملفات `docs` و `results`؟
