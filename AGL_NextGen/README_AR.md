## 🌟 نظرة عامة
مشروع **AGL_NextGen** هو التطور النهائي لنظام الذكاء الاصطناعي العام (AGL). تم تحويل النظام من كود مرتبط ببيئة تطوير محددة إلى **منتج مستقل وقابل للنقل (Portable Product)**. يعتمد النظام على معمارية "Ancestor-based Root Detection" التي تكتشف مسار المشروع تلقائياً، مما يلغي الحاجة للمسارات الصلبة (Hardcoded Paths).

## 🚀 التطورات الرئيسية في الإصدار 2.1.0

### **5. المسار الموحد للذكاء الخارق (Unified ASI Path)**
تم دمج كافة الوحدات في مسار واحد فائق الذكاء. يمكن تشغيل النظام الآن كـ **نظام ذكاء اصطناعي فائق (ASI)** يتعامل مع التفكير الأخلاقي، الحوسبة الكمومية، والبحث المباشر عن المعرفة في آن واحد.

---

## 🛠️ كيفية التشغيل
لإطلاق نظام الذكاء الخارق الموحد (المسار الكامل):
```powershell
python AGL_NextGen_ASI_Launch.py
```

لإجراء فحص شامل للنظام:
```powershell
python .\scripts\AGL_FULL_DIAGNOSTIC.py
```

أو تشغيل دورة الأحلام:
```powershell
python .\src\agl\core\master_controller.py --module dreaming
```
