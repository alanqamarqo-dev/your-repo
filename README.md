## 🇬🇧 English Overview

AGL is an advanced autonomous agent designed to exhibit general intelligence capabilities. Unlike traditional LLM wrappers, AGL possesses:

- **Volition:** Internal drives and goal setting.
- **Self-Evolution:** The ability to read, improve, and hot-swap its own code.
- **Heikal Physics Engine:** Implements the "InfoQuantum Lattice Theory" for advanced problem solving (Tunneling), memory (Gravitational), and logic (Hybrid).
- **Heikal Quantum Security (QSR):** A physics-based ethical phase lock that prevents malicious actions using Quantum Synaptic Resonance.
- **Self-Healing:** Automatic error detection and repair.
- **Dreaming:** Offline memory consolidation and synthetic experience generation.
- **Multi-Domain Competence:** Seamless switching between coding, science, and creative tasks.

### Key Directories

- **`AGL/`**: Core Python package.
- **`repo-copy/Core_Engines/`**: The cognitive modules (Brain).
- **`Integration_Layer/`**: The orchestration logic (Prefrontal Cortex).
- **`Safety_Systems/`**: Ethical and operational safety rails.
- **`docs/`**: Detailed documentation and reports.
- **`HEIKAL_THEORY_INTEGRATION_REPORT.md`**: Documentation of the unified physics theory powering the system.

### Getting Started

1. **Install Dependencies:**

   ```powershell
   pip install -r requirements.txt
   ```

2. **Run the Agent:**

   ```powershell
   python autonomous_agent.py
   ```
